BLOG 1: Crafting a Simple Snake-Water-Gun Game in C: A Month-Long Journey in College First Year


As a budding programmer in my first year of college, I embarked on a month-long coding adventure to create a classic game using the C language.

Week 1: Laying the Foundation


In first week i learned the basic of c like if else statement,loops and made simple programme.

Week 2: Basic Project

In my second week i made my first project and that was GUESSING THE NUMBER.
In this game,computer assign a random number between 1 to 100 and we have to guess it in minimum 10 chance.

Week 3:File Handling and Advanced Topics

To proceed further I learned function,array,pointer etc.
In week 3 I practise what i learned in last 3 week.

Week 4:Building the Snake Water Gun Game

This simple yet entertaining project will require user input, random number generation, and conditional statements.
Snake Water Gun Game Rules:
  • The User have total 5 chance.
  • The user selects either Snake, Water, or Gun.
  • The computer generates a random choice.
  • The winner is determined based on the game's rules: Snake beats Water, Water beats Gun, and Gun beats Snake.
  • Display the result and keep track of the score.
  • If user win 3 or more than 3 chance than user wins the game.

conclusion:

In just a month, I've covered the basics and delved into more advanced concepts of the C language. Building the Snake Water Gun game served as a practical application of your newfound knowledge.
